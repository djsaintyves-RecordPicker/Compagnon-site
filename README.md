# Site public de Compagnon

Site statique minimal pour présenter Compagnon et fournir les URL publiques
nécessaires à sa diffusion :

- accueil et positionnement du produit ;
- politique de confidentialité ;
- assistance et consignes de signalement.

Le site ne charge aucune ressource tierce, ne dépose aucun cookie et n’intègre
aucun outil de mesure d’audience.

## Tester localement

Depuis ce dossier :

```sh
python3 -m http.server 8080
```

Puis ouvrir `http://localhost:8080/`.

## Publication

Le contenu de la branche `main` est publié avec GitHub Pages. Les changements
doivent être relus localement avant publication.

## Données scolaires

Ce dépôt est public. Il ne doit jamais contenir de base EDT réelle, de liste
nominative, de capture identifiable ni d’autre donnée d’établissement.
