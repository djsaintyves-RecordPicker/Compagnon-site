Assistance

Signaler un problème de façon utile, reproductible et respectueuse des données scolaires.

01

Consulter ou ouvrir un ticket

Utilise le dépôt public du site pour les questions générales et les anomalies reproductibles. Vérifie d’abord qu’un ticket similaire n’existe pas.

Accéder aux tickets GitHub

!

Ne publie aucune donnée réelle

Ne joins ni nom d’élève, ni enseignant, ni UAI, ni coordonnées, ni export EDT, ni capture permettant d’identifier un établissement. Utilise uniquement un exemple minimal anonymisé.

02

Informations utiles

  • version de Compagnon et version de macOS ;
  • action exacte réalisée avant le problème ;
  • résultat attendu et résultat observé ;
  • message affiché dans Diagnostic ;
  • possibilité ou non de reproduire le problème.

03

Avant de signaler

  1. crée une sauvegarde Compagnon ;
  2. note l’étape et la configuration concernées ;
  3. vérifie le diagnostic du solveur ;
  4. reproduis, si possible, sur une copie anonymisée.

Disponibilité

Compagnon est actuellement en bêta privée. L’assistance est fournie au mieux pendant la phase de test, sans garantie de délai. Les fonctions de production doivent être utilisées sur des copies et validées avant réimport dans EDT.
